## 📋 部署前必须配置

### 云函数环境变量配置

在微信云开发控制台配置：

1. **打开云开发控制台**
2. **进入云函数列表**
3. **选择需要配置的云函数**：
   - `bill_processor`
   - `ai_advisor`
4. **点击"配置"标签**
5. **添加环境变量**：
   ```
   变量名：GLM_API_KEY
   变量值：你的智谱AI密钥
   ```
6. **保存配置**

### 获取密钥
1. 访问 https://open.bigmodel.cn
2. 注册并登录
3. API Keys → 创建新密钥
4. 复制密钥并配置到云函数
