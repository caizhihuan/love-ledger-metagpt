## 🎨 设计风格

### 色彩方案
```
主色：#FF6B9D（粉色）
辅色：#FF8E9E（浅粉）
成功：#52C41A（绿色）
背景：#F8F8F8（浅灰）
文字：#333333（深灰）
次要文字：#999999（灰色）
```

### 圆角设计
```
- 大卡片：20-24px
- 小卡片：12-16px
- 按钮：24-48px（胶囊形）
- 图标背景：8-16px
```

### 间距规范
```
- 页面边距：16px
- 卡片间距：12-16px
- 元素间距：8-12px
- 文字行高：1.5
```

### 字体大小
```
- 大标题：32-48px
- 中标题：18-20px
- 正文：14-16px
- 小字：10-12px
```
