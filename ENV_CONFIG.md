# 环境变量配置说明

## 云函数环境变量配置

在微信云开发控制台配置以下环境变量：

### GLM_API_KEY（必需）
智谱AI的API密钥，用于：
- 账单图片识别
- 文本智能解析
- AI财务顾问

**获取方式：**
1. 访问 https://open.bigmodel.cn
2. 注册并登录
3. 进入 API Keys 页面
4. 创建新的 API Key
5. 在云开发控制台配置环境变量

**配置步骤：**
1. 打开微信云开发控制台
2. 进入云函数 → 选择云函数
3. 点击"配置"标签
4. 添加环境变量：
   - 变量名：GLM_API_KEY
   - 变量值：你的API密钥
5. 保存配置

### 示例
```
GLM_API_KEY=a550622d7882404c876763f67558f7f1.xxxxx
```

## 安全提醒
⚠️ 永远不要将API密钥硬编码在代码中！
⚠️ 不要将包含密钥的文件提交到Git！
⚠️ 定期更换API密钥！
